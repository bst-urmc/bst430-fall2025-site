Packages

Again, we use the tidyverse, vroom, readxl and janitor packages. These packages is already installed for you. You can load them by running the following in your Console:

library(tidyverse)
library(vroom)
library(readxl)
library(janitor)

Data

We can load the data with the following:

crashes = vroom("https://urmc-bst.github.io/bst430-fall2021-site/hw_lab_instruction/hw02-accidents/data/ny_collisions_2018_2019.csv.gz")

You can find out more about the dataset in the NY open data portal: https://data.ny.gov/Transportation/Motor-Vehicle-Crashes-Case-Information-Three-Year-/e8ky-4vqe . Theres a detailed data dictionary here.
